# Product Overview

Next.js と Server Actions を活用した、型安全なフォーム処理と認証機能を備えたフルスタック Web アプリケーション。日本語 UI を持ち、プロフィール管理を中心としたユーザー向けサービスの基盤となる。

## Core Capabilities

- **認証システム**: Better Auth による Email/Password + GitHub OAuth のマルチプロバイダー認証
- **型安全なフォーム処理**: Conform + Zod による Server Action ベースのフォームバリデーション（サーバー・クライアント両対応）
- **プロフィール管理**: 認証済みユーザーのプロフィール CRUD 操作
- **ダークモード対応**: next-themes によるシステム設定連動テーマ切替

## Target Use Cases

- 認証付き Web アプリケーションの基盤構築
- Server Actions パターンによるフォーム処理の実装リファレンス
- Conform + Zod を用いたプログレッシブエンハンスメント対応フォームの実装例

## Value Proposition

- React 19 の `useActionState` と Conform を組み合わせた最新のフォーム処理パターン
- サーバーサイドとクライアントサイドで同一の Zod スキーマを共有する型安全な設計
- Better Auth + Drizzle ORM による認証・データベース層の型安全な統合
